
# Tae Hyun Kim — Minimal Research Website (Static)

This is a minimal, clean, bilingual (KR/EN) static site for GitHub Pages.

## Quick Deploy
1. Create a new public repo named `yourusername.github.io` on GitHub
2. Upload all files in this folder to the repo root
3. (Optional) Add your `cv/TaeHyunKim_CV.pdf`
4. Settings → Pages → Build and deployment: `Deploy from branch` (main) → Save
5. Your site will be live at `https://yourusername.github.io`

## Customize
- Edit texts in `index.html`, `research.html`, etc.
- Replace `assets/profile.svg` with your photo (e.g., `profile.jpg`) and update the `<img>` path
- Update contact links in `contact.html`
